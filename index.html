Olá, sou o Levy - Dev Full-Stack

Sou um desenvolvedor apaixonado por construir soluções robustas e eficientes. Com experiência tanto no back-end, utilizando Node.js e NestJS para criar arquiteturas escaláveis, quanto no front-end com React, meu foco é transformar ideias em software de alta qualidade, aplicando sempre boas práticas e código limpo.

Back-End

No Back-End, construo APIs com NestJS aplicando boas práticas de arquitetura. Busco seguir os princípios SOLID e prezo por um código limpo e coeso, facilitando a escalabilidade e a manutenção dos sistemas.

Banco de Dados

Experiência em modelagem e implementação de bancos de dados relacionais como PostgreSQL e MySQL para garantir a integridade e a estrutura dos dados.

Front-End

No Front-End, minha paixão é criar interfaces intuitivas e experiências de usuário fluidas com React. Busco utilizar as melhores práticas de componentização e gerenciamento de estado com hooks para construir SPAs (Single Page Applications) que sejam tanto performáticas quanto agradáveis de usar.
